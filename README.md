# SillyTavern 多候选帮答扩展

这个扩展会在 SillyTavern 输入区旁边增加一个 `多候选帮答` 按钮。

点击后会：

1. 基于当前聊天上下文，按不同风格分别请求模型。
2. 一次生成多条“我下一句该怎么回”的候选回复。
3. 让你把任意候选一键替换进输入框、追加到输入框，或复制出来。

## 已实现功能

- 自定义候选数量
- 自定义生成长度上限
- 自定义文本长度要求
- 自定义风格列表
- 自定义额外要求
- 可选把输入框现有内容当作草稿/意图参考
- 结果面板内直接选择候选并插入输入框

## 当前目录

当前成品目录：

`D:\SillyTavern-MultiReply-Assistant`

安装到酒馆时，建议保留文件夹名不变。

## 安装方式

### 方式 1：安装为全局第三方扩展

把整个文件夹复制到：

`<你的 SillyTavern 目录>\public\scripts\extensions\third-party\SillyTavern-MultiReply-Assistant`

### 方式 2：安装为当前用户扩展

把整个文件夹复制到：

`<你的 SillyTavern 目录>\data\<你的用户目录>\extensions\SillyTavern-MultiReply-Assistant`

## 安装后怎么用

1. 重启 SillyTavern，或刷新前端页面。
2. 打开任意聊天。
3. 在输入区附近点击 `多候选帮答`。
4. 在右侧面板里调整参数后点 `开始生成`。
5. 从候选卡片里选择一条，直接插入输入框。

## 参数说明

- `候选数量`：一次生成几条不同风格回复。数量越大，请求次数越多。
- `生成长度上限`：调用 SillyTavern quiet prompt 时使用的最大回复长度。
- `文本长度要求`：自然语言长度约束，例如“每条 30-60 字”。
- `风格列表`：每行一个风格，扩展会按顺序生成。
- `额外要求`：你自己的额外规则，例如“别太油”“别用网络梗”。
- `把输入框现有内容作为草稿/意图参考`：会把你当前已经写在输入框里的内容一起喂给帮答提示词，方便润色改写。

## 兼容说明

- 基于 SillyTavern UI Extension 机制实现。
- 使用 `generateQuietPrompt` 进行后台生成。
- 每个候选会单独请求一次模型，所以会比单次普通帮答更耗时、更耗 token。

## 文件结构

- `manifest.json`：扩展清单
- `index.js`：主逻辑
- `style.css`：样式
- `settings.html`：设置页 UI
